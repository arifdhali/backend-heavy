# Backend Heavy 🚀

Backend Heavy is my personal system design and backend engineering practice repository.

Here, I build real-world backend features from scratch and test how they behave as the number of users grows from a few users to thousands and eventually millions.

## What You'll Find Here

* Notification Systems (0 → 1M Users)
* Instagram-style Follow/Unfollow Systems
* User Feed Generation
* Database Relationship Design
* Scalability Experiments
* High Volume Data Insertion
* Queue & Background Job Concepts
* API Design & Optimization
* System Design Implementations
* Performance Testing

## Why This Repository?

Instead of only learning theory, I implement backend systems and analyze:

* How databases perform at scale
* How large applications handle millions of users
* Different approaches to system design problems
* Trade-offs between simplicity and scalability
